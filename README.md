# google-tunix-hack

My submission for the [Google Tunix Hackathon](https://www.kaggle.com/competitions/google-tunix-hackathon) on Kaggle.

## About the Competition

**Google Tunix Hack - Train a Model to Show Its Work**

This hackathon, hosted by Google Cloud, challenges participants to train Gemma models to show reasoning using Tunix, Google's new JAX-native library for LLM post-training.

### Challenge Goal

Create a model that not only provides the correct answer but also explains the steps it took to get there, making AI more capable, trustworthy, and transparent.

### Requirements

- Create a working training pipeline using Tunix
- Use either Gemma2 2B or Gemma3 1B model
- Submit a Kaggle Writeup, public notebook, and video

### Timeline

- **Launch Date:** November 12, 2025
- **Final Submission:** January 12, 2026
